Layout

This is for laying out items on a page.

This template is using a 2 column layout with a sidebar (3cols) and main (9cols). View the source.

The default grid system is 12 columns but we added some padding/margin to the main container which really makes this for us an 8 column layout. Shouldn't be a big deal.
